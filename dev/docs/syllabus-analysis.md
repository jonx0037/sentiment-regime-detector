## Paper Requirements

### Length & Format

- **Minimum**: 20 pages
- **Template**: Must use provided template
- **Target Audience**: General technical audience (think MSDS students just beginning the program)

### Required Sections

1. **Abstract**: 200-word executive summary
2. **Introduction**: 2-4 pages executive summary of the paper
3. **Literature Review**: Sufficient information about problem domain and relevant prior work
4. **Data**: Well-described attributes and collection methods
5. **Methods/Solution**: Detailed enough for reproduction by knowledgeable readers
6. **Results**: Clear presentation with readable tables and figures
7. **Discussion**: Meaning and application of your solution
8. **Ethics**: At least one page covering potential ethical issues
9. **Conclusions**: Supported by analysis (not just summary)
10. **References**: Proper citations (before periods, next to cited words)

### Quality Standards

- Publication-ready quality
- Good grammar and correct formatting
- Coherent, sequential story without missing steps
- Compelling narrative that holds reader interest
- All citations properly formatted
- No plagiarism (referral to Honor Council results in 'F')

## Code Submission Requirements

### Repository Structure

```text
capstone-project/
├── README.md (execution instructions)
├── data/ (or instructions to acquire)
├── src/ (well-organized source code)
├── notebooks/ (analysis notebooks)
├── requirements.txt (dependencies)
└── docs/ (additional documentation)
```

### Code Quality Standards

- Clean, well-commented code
- Reproducible results
- Clear documentation
- Proper version control
- Dependencies clearly listed

## Important Policies

### Collaboration & Sharing

- All assignments must be shared with professor via OneDrive/Office 365
- Professor must have editing privileges for feedback
- Use SMU email for sharing (not personal accounts)

### Attendance & Participation

- Must attend Immersion for final presentation
- All group members must present
- Active participation expected in peer reviews

### Academic Integrity

- Zero tolerance for plagiarism
- Proper citation required for all sources
- Any violation results in course failure
